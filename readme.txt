How to setup project:
	1) Download the virtual box image at: https://drive.google.com/file/d/1eAjS5yD_xAgdTqLHzk1hGUCHfGeaN6jq/view?usp=sharing
	2) Unzip the file 
	3) Download Vritual box and select "Add" form the machine Drop down menu
	4) Select the .vbox file and press "Open"
	5) Start the virtual machine
	6) The password to the VM is 12345
	5) Type "localhost/" into the webbrowser to get to the main page of the website.
